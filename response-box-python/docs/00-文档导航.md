## 💡 学习建议

### 初学者路径（推荐）
1. 阅读 [01-快速入门指南](01-快速入门指南.md) 完整内容
2. 运行 `main.py` 测试反应盒
3. 参考 `experiment_psychopy.py` 编写自己的实验
4. 有问题时查阅 [02-完整使用手册](02-完整使用手册.md)

### 研究者路径
1. 快速浏览 [01-快速入门指南](01-快速入门指南.md)
2. 详读 [03-时间参数详解](03-时间参数详解.md)
3. 学习 [04-误差分析与校准](04-误差分析与校准.md)
4. 运行系统延迟校准测试
5. 按照第6章要求报告方法部分

### 开发者路径
1. 查看 [PROJECT_STRUCTURE.md](PROJECT_STRUCTURE.md) 了解代码组织
2. 阅读 `src/` 模块的源代码
3. 参考 `experiment_psychopy.py` 进行二次开发

## 🔧 代码示例

### 命令行测试
```bash
uv run main.py
# 或
python main.py
```

### PsychoPy实验
```bash
uv run experiment_psychopy.py
# 或
python experiment_psychopy.py
```

### 自定义实验
```python
from src import auto_find_reaction_box, get_reaction_time

PORT = auto_find_reaction_box()
result = get_reaction_time(PORT)
print(f"反应时: {result['t1']} ms")
```
