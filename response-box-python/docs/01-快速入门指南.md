**关键概念**：
1. **串口通信基础**
   - 波特率：115200
   - 超时时间：可配置
   - 字节流传输

2. **命令协议**
   ```python
   # 查询设备
   ser.write(b'\x5A\x00')
   
   # 停止计时
   ser.write(b'\xFC\x00')
   
   # 触发计时（重点！）
   ser.write(b'\xFB\x00')
   ```

3. **数据帧格式**
   ```
   [FB] [按键] [t1高] [t1低] [t2高] [t2低] [00] [01]
   ```

### 第5步：了解误差和校准（可选，30分钟）

**学习材料**：
- 📄 [反应时参数与误差分析.md](反应时参数与误差分析.md) - 第3章

**关键概念**：
1. **系统误差**：
   - 串口延迟：0.5-2 ms
   - 硬件延迟：< 1 ms
   - **总共约 2-3 ms**

2. **测量误差**：
   - 时钟精度：± 0.5 ms
   - 按键抖动：± 1-2 ms

3. **是否需要校准？**
   - 如果实验对比不同条件：**不需要**（系统误差对所有条件相同）
   - 如果需要绝对准确值：**需要校准**

```python
"""
简单的Stroop实验示例
"""
from psychopy import visual, core
from 反应盒测试示例 import get_reaction_time, auto_find_reaction_box
import random

# 1. 初始化
PORT = auto_find_reaction_box()
win = visual.Window(size=(800, 600), color='white')

# 2. Stroop刺激
stimuli = [
    {'word': '红色', 'color': 'red', 'congruent': True},
    {'word': '绿色', 'color': 'green', 'congruent': True},
    {'word': '红色', 'color': 'green', 'congruent': False},
    {'word': '绿色', 'color': 'red', 'congruent': False},
]

# 3. 指导语
instruction = visual.TextStim(win, 
    text='看到红色字按红键(3)\n看到绿色字按绿键(1)\n\n按任意键开始',
    color='black')
instruction.draw()
win.flip()
get_reaction_time()  # 等待按键

# 4. 实验循环
results = []
for trial in range(12):  # 12个试次
    # 随机选择刺激
    stim = random.choice(stimuli)
    
    # 呈现刺激
    text = visual.TextStim(win, text=stim['word'], 
                          color=stim['color'], height=0.2)
    text.draw()
    win.flip()
    
    # 获取反应
    key, rt, press, _ = get_reaction_time(port=PORT)
    
    # 判断正误
    correct_key = 2 if stim['color'] == 'red' else 1
    correct = (key == correct_key)
    
    # 记录数据
    results.append({
        'trial': trial + 1,
        'word': stim['word'],
        'color': stim['color'],
        'congruent': stim['congruent'],
        'key': key,
        'rt': rt,
        'correct': correct
    })
    
    # 反馈
    feedback = visual.TextStim(win, 
        text='正确!' if correct else '错误!',
        color='green' if correct else 'red')
    feedback.draw()
    win.flip()
    core.wait(0.5)

# 5. 保存数据
import pandas as pd
df = pd.DataFrame(results)
df.to_csv('stroop_results.csv', index=False)
print(f"正确率: {df['correct'].mean()*100:.1f}%")

win.close()
```

**检验标准**：
- [ ] 能将反应盒整合到PsychoPy实验中
- [ ] 能在刺激呈现时触发计时
- [ ] 能正确记录和保存数据
- [ ] 能根据按键判断反应正误
